Research Framework Notes

Primary Case (Harvey): Serves as archetypal event with extensive existing documentation (Zhai 2025, FEMA records, media coverage). Use for detailed resident mapping, interviews, and establishing baseline neighborhood perception patterns.

Shadow Cases: Seven additional events provide comparative context and test specific hypotheses:

  • Memorial Day 2015 & Tax Day 2016: Pre-Harvey events establish temporal sequence; test whether early floods "primed" neighborhoods for Harvey impacts
  • Imelda 2019: Tests cumulative vulnerability - do neighborhoods hit by both Harvey and Imelda show amplified perception changes?
  • Winter Storm Uri 2021: Different hazard type; tests whether non-flood disasters produce similar income-stratified recovery patterns
  • Heat Events 2023-24: Chronic vs. acute stress; "slow violence" of persistent climate pressure vs. dramatic flood events
  • Derecho 2024: Most recent event; allows real-time perception mapping with less memory distortion

Analytical Questions Across Cases:

  • Geographic Concentration: Do the same neighborhoods get hit repeatedly (cumulative vulnerability) or do different events affect different areas?
  • Damage Comparability: Can you identify neighborhood pairs with similar physical exposure but different socioeconomic characteristics?
  • Recovery Speed: Does recovery timeline differ by income even when damage severity is comparable?
  • Perception Persistence: Do older events (Memorial Day 2015) still shape current neighborhood boundaries, or does perception reset after each event?
  • Event Type Matters: Do floods produce different perception changes than cold events or heat events?

Data Collection Strategy: For Harvey, conduct full resident mapping + interviews. For shadow cases, use existing secondary sources (FEMA declarations, news reports, city assessments) to establish patterns. If rich data sources emerge for specific shadow cases (e.g., strong community organization documentation of Uri impacts), expand analysis of those events.

Cumulative Vulnerability Index: Create composite measure: # of events affecting each neighborhood × severity of damage in each. This allows comparing "frequently hit" vs. "occasionally hit" neighborhoods with similar total damage levels.
